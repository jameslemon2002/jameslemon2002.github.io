# Rong Zhao GitHub Pages Site

This folder is a complete static GitHub Pages site.

To publish:

1. Create or open the repository `jameslemon2002.github.io`.
2. Copy `index.html` and the `assets/` folder to the repository root.
3. Commit and push to the `main` branch.
4. The site should be available at `https://jameslemon2002.github.io/`.
